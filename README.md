# Cabinet TCHOUHO - Frontend

## Vue d'ensemble

Le site vitrine du Cabinet TCHOUHO est une application moderne développée avec React et Vite. Il offre une interface utilisateur fluide et responsive pour présenter les services juridiques du cabinet.

## Technologies principales

- React
- Vite
- Tailwind CSS
- TypeScript

## Installation

1. Clonez le dépôt :
   ```bash
   git clone <URL-du-dépôt>
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```

## Développement

Lancez le serveur de développement :
```bash
npm run dev
```

## Déploiement

L'application est déployée sur Vercel. Pour un déploiement manuel :
```bash
npm run build
```
